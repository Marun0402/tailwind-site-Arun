Getting Started is Easy

Follow our quick start guide to learn how to call our APIs and maximise the use of our microservices.

  1. What are the different types of microservices offered?
  2. How do I call an API?
  3. What is a Demo App?

What are the different types of microservices offered?

We offer 5 different categories of microservices:

Voice -for processing audios, including most Asian languages

Natural Language Processing -for processing language-related tasks

Computer Vision - for processing photos, videos and livestreams

Data - available as query-able data APIs or download service APIs

Utility - non AI microservices that are added conveniences for developers

How do I call an API?

Sign up for a free account at [URL]

Choose an API Endpoint

Explore our API documentation to understand the range of microservices we offer. Subscribe to the microservice that aligns best with your desired functionality and obtain the API endpoint.

Construct an HTTP request

Using a HTTP verb such as GET, POST, PUT or DELETE, construct the HTTP request to call the API endpoint of your choice. Include the API key and send the request to the API endpoint using your preferred programming language or tool.

What is a Demo App?

A demo app is a single-page web application that demonstrates how an API may be used in an end-user application. A demo app may often utilise more than one microservice to demonstrate how our microservices can work together.

You may download the source code of the demo app to see how the app calls the API from the Sentient.io platform and you may also use the source code as a reference to build your own application.
